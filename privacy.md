# Privacy Policy - Qawi AI & Kenra Project

**Last Updated:** March 2026  
**Provider:** Vanguard Biz

## 1. Data Collection
Qawi AI only processes data explicitly provided by the user through Discord commands or interactions. This includes:
* Security reports and extortion-related alerts.
* Public Discord User IDs for role assignment (PoB).

## 2. Use of Data
The collected data is used exclusively for:
* Generating security logs and intelligence reports within the Kenra ecosystem.
* Creating cryptographic hashes for traceability on the **zkTanenbaum (zkSYS)** network.

## 3. Blockchain Transparency
Users acknowledge that while personal data is not stored on-chain, a permanent, immutable hash of the security report will be recorded on the Syscoin blockchain for forensic integrity.

## 4. Third-Party Services
Qawi AI operates as an autonomous proxy. We do not sell or share user data with third parties, except for automated processing through our secure AI engine.

## 5. Security
Vanguard Biz implements industry-standard encryption to protect all interactions between the Discord Gateway and our autonomous agents.
